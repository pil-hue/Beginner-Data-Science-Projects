# Beginner Data Science Projects
A collection of beginner data science projects built using Python libraries and machine learning fundamentals.

## Projects:
### 1. [Student Performance Analysis](./Student_Performance_Analysis)
An exploratory data analysis project focused on identifying which student-related factors have the strongest relationship with exam scores.

### 2. [Spotify Songs Analysis](/.Spotify_Songs_Analysis)
An exploratory data analysis project examining how acoustic features relate to song popularity across different music genres, as well as how energy is distributed among different genres.
