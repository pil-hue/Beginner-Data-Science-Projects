# Student Performance Analysis
## Objective: The goal of this project was to determine which factor has the strongest relationship with a student's final exam performance using a dataset of 200 students.

## Dataset Features:
- Hours Studied
- Sleep Hours
- Attendance Percentage
- Previous Scores
- Exam Score

## Methodology
- Summarizing the dataset using measures such as averages, spread, and distribution.
- Visualizing the relationship between each factor and the final exam score using scatter plots.
- Measuring the strength of the linear relationship between variables using Pearson's correlation coefficient.
- Comparing the correlation values of all factors visually using a bar chart.

## Key Findings
- Hours studied showed the strongest positive correlation (0.77) with exam scores.
- Previous scores showed a moderate positive relationship (0.43).
- Sleep hours and attendance percentage showed relatively weak relationships with exam scores.
